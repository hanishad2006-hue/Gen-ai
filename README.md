Student Assignment Evaluation System with AI

An AI-powered full-stack web application for managing and evaluating student assignments with role-based dashboards (Student, Professor, HOD, Admin).

This system allows secure authentication, assignment submission, AI-based evaluation, and structured academic workflow management.

Project Overview

This project is designed to:
 Allow Students to submit assignments
 Allow Professors to review & evaluate assignments
 Allow HOD to monitor department performance
 Provide Admin with full system control
Integrate AI-based assignment evaluation
